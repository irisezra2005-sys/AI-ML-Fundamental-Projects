
# AI & ML Fundamentals Project

A structured research project covering the fundamentals of Artificial Intelligence and Machine Learning, a real-world industry use case, and the key ethical considerations in responsible AI development.

##  Project Overview

This project was completed in three parts:

1. **Fundamentals Report** — Core definitions of AI vs. ML, Narrow AI vs. General AI, and an overview of the three primary types of Machine Learning with example algorithms and use cases.
2. **Use Case Analysis** — A deep dive into how the **Healthcare** industry applies AI/ML to solve a real-world problem (early disease detection via medical imaging).
3. **Ethics Note** — An analysis of key ethical implications of AI, including algorithmic bias, data privacy, and responsible AI development practices.
